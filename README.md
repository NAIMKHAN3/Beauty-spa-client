# Getting Started

Clone this repo

```
git clone "Repo Link"
```

Change directory to BD-Tour-Client

```
cd BD-Tour-Client
```

Install the dependencies

```
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
